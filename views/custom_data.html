O que é JSON e como usar?
O que é JSON?

JSON (JavaScript Object Notation) é um formato leve de troca de dados. É fácil para humanos ler e escrever e fácil para máquinas analisar e gerar.

JSON é usado para armazenar e transmitir dados estruturados, como informações de pessoas, produtos, eventos, etc.
Estrutura Básica
Exemplo de JSON
{
    "nome": "João Silva",
    "idade": 30,
    "email": "joao@email.com"
}

Explicação:

  • Chaves {} delimitam um objeto
  • Dados são organizados em pares "chave": "valor"
  • Cada par é separado por vírgula
  • Chaves sempre são textos entre aspas duplas
